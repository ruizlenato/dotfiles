# dotfiles

## i3-WM

![Screenshot](https://github.com/ruizlenato/dotfiles/blob/main/Images/i3wm.png)

| Dependency                                                                         | Description                                                 |
| ---------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| `i3-wm`                                                                            | Is a dynamic tiling window manager                          |
| `polybar`                                                                          | Is a fast and easy-to-use tool for creating status bars.    |
| `nitrogen`                                                                         | Is a desktop background browser and setter for X Window.    |
| `dmenu`                                                                            | Is a fast and lightweight dynamic menu for                  |
| `tiramisu`                                                                         | Is a notification daemon for *nix desktops                  |
| `herbe`                                                                            | Daemon-less notifications without D-Bus.                    |
| `st`                                                                               | Is a simple terminal implementation for X                   |
| `xfce-polkit`                                                                      | A simple PolicyKit authentication agent                     |
